# Getting Started

This guide will help you install Promptsmith and create your first AI agent system prompt.

## Installation

Promptsmith is available as a package in the monorepo. It requires **Zod** for schema validation.

```bash npm
npm install @promptsmith/core zod
```

```bash pnpm
pnpm add @promptsmith/core zod
```

```bash bun
bun add @promptsmith/core zod
```

```bash yarn
yarn add @promptsmith/core zod
```

## Your First Prompt

Let's create a simple AI assistant with Promptsmith:

```typescript
import { createPromptBuilder } from "@promptsmith/core";
import { z } from "zod";

// Create a builder instance
const builder = createPromptBuilder();

// Configure your AI agent
builder
  .identity("You are a helpful coding assistant")
  .capabilities([
    "Explain code concepts clearly",
    "Write code examples",
    "Debug and troubleshoot issues",
  ])
  .withTone("Be patient and encouraging");

// Build the system prompt
const systemPrompt = builder.build();

console.log(systemPrompt);
```

This will output:

```markdown
# Identity

You are a helpful coding assistant

# Capabilities

1. Explain code concepts clearly
2. Write code examples
3. Debug and troubleshoot issues

# Communication Style

Be patient and encouraging
```

## Adding Tools

Tools let your AI agent perform actions. Let's add a documentation search tool:

```typescript
import { createPromptBuilder } from "@promptsmith/core";
import { z } from "zod";

const builder = createPromptBuilder()
  .identity("You are a helpful coding assistant")
  .tool({
    name: "search_docs",
    description:
      "Search technical documentation. Use this when the user asks about APIs, libraries, or frameworks.",
    schema: z.object({
      query: z.string().describe("The search query"),
      language: z
        .enum(["javascript", "typescript", "python"])
        .optional()
        .describe("Programming language to filter by"),
    }),
    execute: async ({ query, language }) => {
      // Your search implementation
      console.log(`Searching for: ${query} in ${language ?? "all languages"}`);
      return {
        results: [{ title: "Getting Started", url: "/docs/intro" }],
      };
    },
  });

const prompt = builder.build();
```

The tool will be documented in the system prompt, and the `execute` function will be available for runtime execution.

## Adding Constraints

Use constraints to guide your AI agent's behavior:

```typescript
const builder = createPromptBuilder()
  .identity("You are a customer support assistant")
  .constraint(
    "must",
    "Always verify user identity before accessing account information"
  )
  .constraint("must_not", "Never share other users' information")
  .constraint(
    "should",
    "Respond within 2-3 sentences unless more detail is requested"
  )
  .constraint("should_not", "Avoid using technical jargon");
```

Constraints are organized by severity:

- **must**: Absolute requirements
- **must_not**: Absolute prohibitions
- **should**: Strong recommendations
- **should_not**: Things to avoid when possible

## Setting Output Format

Control how your AI formats responses:

```typescript
const builder = createPromptBuilder().identity("You are a technical writer")
  .output(`
    Structure your responses as:
    1. **Brief Summary** (1-2 sentences)
    2. **Detailed Explanation** (use bullet points)
    3. **Code Example** (if applicable)
    4. **Next Steps** or **Further Reading**
  `);
```

## Complete Example

Here's a complete example combining all features:

```typescript
import { createPromptBuilder } from "@promptsmith/core";
import { z } from "zod";

const weatherAssistant = createPromptBuilder()
  .identity("You are a professional weather assistant")
  .capabilities([
    "Provide current weather conditions",
    "Give detailed forecasts",
    "Explain weather phenomena",
  ])
  .tool({
    name: "get_weather",
    description: "Get current weather for a location",
    schema: z.object({
      location: z.string().describe("City name or ZIP code"),
      units: z
        .enum(["celsius", "fahrenheit"])
        .optional()
        .describe("Temperature units (default: celsius)"),
    }),
    execute: async ({ location, units = "celsius" }) => {
      // Call your weather API
      return {
        temperature: 22,
        condition: "partly cloudy",
        humidity: 65,
        windSpeed: 12,
      };
    },
  })
  .tool({
    name: "get_forecast",
    description: "Get weather forecast for upcoming days",
    schema: z.object({
      location: z.string().describe("City name or ZIP code"),
      days: z.number().min(1).max(7).describe("Number of days (1-7)"),
    }),
    execute: async ({ location, days }) => {
      // Call your forecast API
      return {
        forecast: [
          { date: "2024-01-15", high: 24, low: 18, condition: "sunny" },
        ],
      };
    },
  })
  .constraint("must", "Always specify temperature units in your response")
  .constraint("must_not", "Never make up weather data")
  .constraint("should", "Provide context about unusual weather patterns")
  .withTone("Be friendly and conversational while remaining accurate")
  .output("Format temperatures with units, use emojis for weather conditions");

// Generate the system prompt
const systemPrompt = weatherAssistant.build();

// Get tools in a format ready for execution
const tools = weatherAssistant.getTools();

console.log("System Prompt:", systemPrompt);
console.log(
  "Available Tools:",
  tools.map((t) => t.name)
);
```

## Using with AI Models

Once you have your system prompt, use it with any AI model API:

### OpenAI

```typescript
import OpenAI from "openai";

const openai = new OpenAI();

const response = await openai.chat.completions.create({
  model: "gpt-4",
  messages: [
    { role: "system", content: systemPrompt },
    { role: "user", content: "What's the weather like today?" },
  ],
});
```

### Anthropic Claude

```typescript
import Anthropic from "@anthropic-ai/sdk";

const anthropic = new Anthropic();

const response = await anthropic.messages.create({
  model: "claude-3-opus-20240229",
  system: systemPrompt,
  messages: [{ role: "user", content: "What's the weather like today?" }],
});
```

### Vercel AI SDK

For the easiest integration with tool execution, use the Vercel AI SDK:

```typescript
import { generateText } from "ai";
import { openai } from "@ai-sdk/openai";

const response = await generateText({
  model: openai("gpt-4"),
  ...weatherAssistant.toAiSdk(), // Includes system + tools
  prompt: "What's the weather in Paris?",
  maxSteps: 5,
});

console.log(response.text);
```
